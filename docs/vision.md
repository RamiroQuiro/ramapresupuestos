\# Visión



ramaPresupuesto es una herramienta web que permite a cualquier profesional crear y compartir presupuestos profesionales en menos de un minuto.



El objetivo no es reemplazar un ERP.



El objetivo es responder rápidamente a un cliente.



El producto será mobile-first.



Será gratuito con un plan PRO.



Forma parte del ecosistema RamaTools.
